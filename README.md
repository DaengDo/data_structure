# 자료구조 학습 일지

```
소프트웨어 업계에서 평생 밥 벌어 먹으면서 살고 싶습니다.
그런데 컴퓨터가 자료를 어떻게 처리하는지 조차 모르고 있네요?
작동 원리가 어떻게 실제로 사용되는지 알기 위해 자료구조를 좀 살펴봅니다.
```

## 배열

ADT

- `<인덱스, 값>` 쌍의 집합
- `create(size)`: size개의 요소를 저장할 수 있는 배열 저장
- `get(A, i)`: 배열 A의 i번째 요소 반환
- `set(A, i, v)`: 배열 A의 i번째 위치의 값을 v로 저장
